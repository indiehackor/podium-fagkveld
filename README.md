### layout-server
Podium layout that fetches and stitch together all the podlet into one page.
### vue-sidebar
Vue podlet

### react-main
React podlet
### html-footer
Plain html and css podlet
### html-header
Plain html and css podlet

## How to run
1. `npm install`
2. `npm run all`

Then visit `http://localhost:8080`

Or you could run each podlet individually from its directory.
